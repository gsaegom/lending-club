# Automated Decision-Making Process - Learning Club

The goal of this project is to automate the decision-making process for LendingClub, an American financial services company. This project has four parts, based on the feature to predict: accepted-rejected (whether a loan is approved or not) and, for the approved loans, grade, sub-grade, and interest-rate. Each project can be found in its corresponding package, as well as the relevant models.

The database used is based on https://storage.googleapis.com/335-lending-club/lending-club.zip. It has been turned into a database, with views, and some column modifications. It can be found in database.zip (file compressed for size reasons).

An Flask application of this project has been uploaded to Google Cloud. This is the URL https://fastapi-project-2-efkrkcyoza-no.a.run.app/docs.

You can use these examples to test the application:
